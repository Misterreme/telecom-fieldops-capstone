Proyecto: FieldOps Telecom Suite (capstone)

Objetivo: construir un sistema de operaciones y ventas para una empresa de telecomunicaciones, con soporte offline, auditoría, seguridad a nivel de código, API First, y dashboard con KPIs.

Entregables obligatorios: OpenAPI actualizado, RF/RNF/RB, edge cases, threat model, audit events, diagramas, pruebas, logs, manejo de errores, y hosting.


Restricciones: no links externos a assets; offline-first con LocalStorage + export/import; seguridad aplicada por código.
